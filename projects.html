Projects

I've worked on a variety of projects, but most are not publically available. Here are a few work samples I can share, though!

Illustro

A responsive, translation-ready WordPress theme built using Bootstrap 3.

Illustro is a responsive WordPress theme based on Bootstrap-3.3.7. Illustro is the first theme I built as part of my WordPress Theming 2 class at PCC, back in 2016. Features include page and post templates, advanced custom fields and options, hero images, a jumbotron area, sidebars and widget areas, and more.

Check out the project over on Github

Recommended Summer Reading List

A dynamic list of book I've read, built using Vue.js, Axios, and the Google Books API.

Recommended Summer Reading List is a proof-of-concept I built as part of my emerging web tools and trends class at PCC. I built the project in 2017, and it was my first experience using Axios and in general, working with APIs. It's also the first time I worked with Vue.js, and I really enjoyed it! I'd love to revisit this project at some point to turn it into a full-fledged app, where folx could enter their own books and receive custom recommendations. For now, though, it's just a dynamic list of books I enjoy, with a custom search feature that pulls data from Google Books.

Check out the project over on Github
